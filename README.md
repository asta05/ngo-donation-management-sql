# NGO Donation & Fund Management System (SQL)

## Objective

Designed and analyzed a database system to manage donors, donations, projects, and beneficiaries for an NGO.

## Tools Used

MySQL

## Database Design

* Created tables for donors, donations, projects, and beneficiaries
* Established relationships between tables using keys

##  Key Operations

* Performed data analysis using SQL queries
* Used JOINs to combine multiple tables
* Applied GROUP BY for aggregation
* Created views for simplified analysis
* Implemented stored procedures and triggers

##  Files Included

* `queries.sql` – SQL queries



##  Sample Queries

* Total donations per project
* Top donors by contribution
* Monthly donation trends
* Beneficiary distribution

##  Key Insights

* Identified top contributing donors
* Analyzed donation patterns across projects
* Improved data organization and consistency
